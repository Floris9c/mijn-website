Project 1

Beschrijving van het project

Dit project gaat dieper in op een onderwerp binnen systeem en netwerkbeheer. Op deze pagina geef ik extra uitleg over de inhoud, de doelstellingen en de manier waarop ik dit project heb uitgewerkt.

Tijdens dit project heb ik gewerkt rond het analyseren van technische onderdelen en het stap voor stap opbouwen van een correcte oplossing. Daarbij heb ik niet alleen gekeken naar de theorie, maar ook naar de praktische uitwerking en de toepassing in een realistische situatie.

Wat heb ik gedaan?

Ik heb eerst onderzocht wat het project precies inhield en welke onderdelen belangrijk waren. Daarna heb ik de verschillende stappen uitgewerkt, getest en waar nodig aangepast. Op die manier kreeg ik een beter inzicht in hoe het volledige project opgebouwd is.

Wat heb ik geleerd?

Door dit project heb ik meer geleerd over technische structuren, het oplossen van problemen en het logisch opbouwen van een werkende oplossing. Daarnaast heb ik ook geleerd om nauwkeuriger te werken en beter na te denken over de opmaak en de presentatie van mijn werk.

Besluit

Dit project heeft mij geholpen om mijn kennis verder uit te breiden en meer ervaring op te doen binnen mijn opleiding. Het was een interessante oefening waarin ik zowel technische als praktische vaardigheden verder heb ontwikkeld.
